# Writing Style Engram — Geoff Trembley

This repo contains my public writing style engram — a structured profile of how I communicate, built from analysis of ~500 messages across chat, email, internal documents, and published articles.

## What Is an Engram?

An engram is a compressed, reusable style profile for AI writing assistants. Drop it into a system prompt and the model writes in your voice — not generic AI prose, not corporate fluff. Yours.

This one captures:

- **Tone & voice** — direct, collegial, operator-minded
- **Sentence structure** — short and declarative with intentional parentheticals
- **Vocabulary** — specific phrases I reach for, and the ones I never use
- **Document patterns** — BLUF-first structure, numbered pillars, labeled asks
- **Thought leadership style** — conversational authority, not academic distance
- **Anti-patterns** — what this voice explicitly does NOT sound like

## How to Use It

Paste the contents of `engram.md` as a system prompt (or prepend it to your first message) in any AI writing assistant — Claude, ChatGPT, Gemini, or similar.

Example system prompt wrapper:

```
You are a writing assistant. Use the following style guide to match my voice exactly.

[paste engram.md here]

Now help me write: [your task]
```

## What It Is Not

This engram contains no proprietary data, company names, customer information, or confidential content. All examples have been anonymized. It describes writing patterns only.
